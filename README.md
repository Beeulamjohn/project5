# 🔑 Modern Login Page (HTML + CSS)

This project is a **login page UI** built using **HTML and CSS**.  
It features a two-column layout with an illustration on the left and a styled login form on the right, including social media sign-in options.  

---

## 📸 Preview
Modern login.png


## 📂 Project Structure

```

project-folder/
│── index.html        # Main HTML file
│── work.css          # CSS stylesheet
│── girl.png          # Illustration image
│── google.png        # Google logo for login
│── apple.png         # Apple logo for login
│── facebook.png      # Facebook logo for login

```

---

## 🚀 Features

- **Two-Column Layout**  
  - Left: Illustration (`girl.png`).  
  - Right: Login form with user inputs and options.  

- **Login Form**  
  - Input fields for **username** and **password**.  
  - "Recovery password" link.  
  - **Sign In button** with orange styling.  
  - Divider with **"Or Continue with"**.  

- **Social Media Login**  
  - Google, Apple, and Facebook login logos.  

- **Clean Layout** with light background (`whitesmoke`) for the form section.  

---

## 🎨 Styling (work.css)

- **Background**: light grey (`rgb(211, 204, 204)`).  
- **Flexbox Layout** for side-by-side image and form container.  
- **Form Styling**: centered content, rounded input fields, padding for spacing.  
- **Button Styling**: orange (`rgb(209, 95, 54)`) with rounded edges.  
- **Typography**:  
  - `h6`: right-aligned "Not a member? Register Now".  
  - `h2` and `h5`: left-aligned greetings.  
  - `p`: aligned with padding for neat structure.  

---

## 🛠️ How to Run

1. Clone or download the repository.  
2. Place the required images (`girl.png`, `google.png`, `apple.png`, `facebook.png`) in the project folder.  
3. Open `index.html` in any modern browser (Chrome, Firefox, Edge, etc.).  

---

## 📌 Notes

- This is a **static UI only** – no backend authentication is implemented.  
- Replace images with your own branding if desired.  
- Add **media queries** to make the layout responsive for mobile devices.  
- You can integrate real **OAuth login APIs** (Google, Apple, Facebook) for full functionality.
